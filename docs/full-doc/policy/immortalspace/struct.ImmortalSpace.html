Expand description

This type implements a simple immortal collection policy. Under this policy all that is required is for the “collector” to propagate marks in a liveness trace. It does not actually collect.

Fields

mark_state: MarkStatecommon: CommonSpace<VM>pr: MonotonePageResource<VM>vm_space: bool

Is this used as VM space? If this is used as VM space, we never allocate into this space, but we trace objects normally.

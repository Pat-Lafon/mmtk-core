Module mmtk::util::edge_logger

source · [−]
Expand description

Logging edges to check duplicated edges in GC. This is a simple module to log edges and check for duplicate edges.

It uses a hash-set to keep track of edge, and is so very expensive. We currently only use this as part of the extreme_assertions feature.

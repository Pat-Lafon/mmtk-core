Expand description

A GC worker. This part is privately owned by a worker thread. The GC controller also has an embedded GCWorker because it may also execute work packets.

Fields§

§tls: VMWorkerThread

The VM-specific thread-local state of the GC thread.

§ordinal: usize

The ordinal of the worker, numbered from 0 to the number of workers minus one. The ordinal is usize::MAX if it is the embedded worker of the GC controller thread.

§scheduler: Arc<GCWorkScheduler<VM>>

The reference to the scheduler.

§copy: GCWorkerCopyContext<VM>

The copy context, used to implement copying GC.

§mmtk: &'static MMTK<VM>

The reference to the MMTk instance.

§is_coordinator: bool

True if this struct is the embedded GCWorker of the controller thread. False if this struct belongs to a standalone GCWorker thread.

§shared: Arc<GCWorkerShared<VM>>

Reference to the shared part of the GC worker. It is used for synchronization.

§local_work_buffer: Worker<Box<dyn GCWork<VM>>>

Local work packet queue.

fn poll(&self) -> Box<dyn GCWork<VM>>

Poll a ready-to-execute work packet in the following order:

  1. Any packet that should be processed only by this worker.
  2. Poll from the local work queue.
  3. Poll from activated global work-buckets
  4. Steal from other workers
